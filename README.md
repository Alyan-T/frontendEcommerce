## Frontend Ecommerce Store

A modern, responsive frontend implementation of an E-commerce platform. This project focuses on the user interface and shopping experience, featuring a clean design and seamless navigation between product listings and checkout flows.

Features
Dynamic Product Grid: Browsing products with categorized views.

Product Details Page: Detailed views with image galleries and descriptions.

Shopping Cart: Add/remove functionality with real-time price updates (State Management).

Responsive Design: Fully optimized for mobile, tablet, and desktop views.

Interactive UI: Hover effects, smooth transitions, and intuitive navigation.
